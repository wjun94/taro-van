# Cell 单元格

## 介绍

单元格为列表中的单个展示项。

## 引入

```jsx | pure
import { Cell } from 'taro-van';
```

## 代码演示

### 基础用法

```jsx | iframe
/**
 * demoUrl: https://wjun94.github.io/taro-van/dist/#/pages/cell/index
 */
<Cell title='标题' isLink value='内容' />
```

## API

### Button

| 参数   | 说明                           | 类型      | 默认值 |
| ------ | ------------------------------ | --------- | ------ |
| title  | 左侧标题                       | `string ` | -      |
| value  | 右侧内容                       | `string`  | -      |
| extra  | 标题下方的描述信息             | `string`  | -      |
| isLink | 是否展示右侧箭头并开启点击反馈 | `boolean` | false  |
